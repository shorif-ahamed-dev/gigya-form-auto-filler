# Form Auto Filler Chrome Extension

A Chrome extension that automatically fills form fields with default values when you click the extension icon.

## Features

- Fills email, first name, last name, and password fields
- Selects country (USA) and language (English) dropdowns
- Checks required checkboxes (age confirmation and terms)
- Simple one-click operation
